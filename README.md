# MIT-Analytics-Edge
This coursework follows edX Analytics Edge MOOC offered by MIT.  Familiarity with R will be useful, but isn't required as a prerequisite.

## Topics Covered
- R
- Linear Regression
- Logistic Regression
- Trees
- Clustering
- Visualization
- Linear Optimization
- Integer Optimization
- Kaggle Competition

## Course timeframe
April 2016 - June 2016
